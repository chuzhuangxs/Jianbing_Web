我和我男朋友两个人都不会省钱，有钱的时候就不停地消费，我不知道我们俩什么时候才能长大。我们的房租三个月交一次，每次到了要交房租的时候，我才会发现：哎，怎么又没钱了……郁闷！什么时候才能学会有计划的消费呢？狂郁闷…… 我手机停机了，但我都不想去充卡了。每天电话接着好烦的，没事还不如多看看书。
